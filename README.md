# AoriHagemashiBot
このBotはDiscord上で動作します。

ユーザーが発言する度に煽りと励ましを一定の確率で返します。

## このBotの仕様

**デフォルトの煽り・励ましメッセージの対象者**

* このBotを作った時に受験生を対象に作って欲しいと頼まれたため、受験生を対象とした煽り・励ましメッセージとなっています。

**コマンド**

* /aori_and_hagemasi
  * [start / stop}
    * 煽り・励ましのオン・オフを行います。
